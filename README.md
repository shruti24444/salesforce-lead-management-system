# Salesforce Lead Management System

## Overview
This project implements a Lead Management CRM system in Salesforce to automate lead scoring, categorize leads based on budget, and generate follow-up tasks for sales teams.

## Features
• Automated lead scoring using Record Triggered Flows  
• Lead categorization (Hot / Warm / Cold)  
• Automatic task creation for high-value leads  
• Reports and dashboards for lead analytics  
• Custom Lightning App for Lead Management

## Technologies Used
Salesforce CRM  
Record Triggered Flows  
Reports & Dashboards  
Lightning App Builder  
Validation Rules


## Business Use Case
This system helps sales teams automatically identify high-value leads and prioritize follow-ups using automation and analytics.
